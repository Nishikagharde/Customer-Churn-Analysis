# Customer Churn Analysis Project

## 📊 Overview

This project analyzes customer churn using both **Power BI dashboards** and **Machine Learning (Python)** to identify patterns and predict customer behavior.

---

## 🚀 Key Features

* Interactive Power BI dashboard for churn insights
* Logistic Regression model for churn prediction
* ROC Curve and performance evaluation
* Data preprocessing and feature engineering

---

## 📈 Results

* Accuracy: ~78%
* AUC Score: ~0.82
* Model: Logistic Regression

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Scikit-learn)
* Power BI
* Matplotlib / Seaborn

---

## 📁 Project Files

* `customer_churn_prediction.ipynb` → ML model
* `customer_churn_dashboard.pbix` → Power BI dashboard
* `Churn.csv` → Dataset
* `predictions.csv` → Model output

---

## 📌 Insights

* Month-to-month contracts have highest churn
* Fiber optic users show higher churn rate
* Long-term contracts reduce churn significantly
